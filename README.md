INSTRUCIONES COMANDOS QUE SIEMPRE USARAS

(Windows) set FLASK_APP=app.py

(Cualquier SO) flask run

(Linux/MacOS) export FLASK_ENV=development
(Windows) set FLASK_ENV=development
